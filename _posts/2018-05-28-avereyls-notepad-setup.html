Well. A notepad powered by  Jekyll gathering some information I wanna keep. It actually is a lot easier than I thought it was going to be thanks to jmcglone.

Install Ruby 

Install Ruby with headers and RubyGems (see requirements)

  •   The simplest way is to download and execute installer from here.
  •   Follow first steps of Jekyll official documentation for macOS available here.
  •   Same for Ubuntu (here).

Install Jekyll and Bundler

{% highlight shell %} # Install Jekyll and Bundler gems through RubyGems gem install jekyll bundler # Create a new Jekyll site at ./myblog jekyll new myblog # Change into your new directory cd myblog # Build the site on the preview server bundle exec jekyll serve # Now browse to http://localhost:4000 {% endhighlight %}

Extras

To use pagination

{% highlight shell %} # Install jekyll-paginate-v2 gems through RubyGems gem install jekyll-paginate-v2 {% endhighlight %}

Installing a gem from behind a proxy

{% highlight shell %} set HTTP_PROXY=http://proxyServer:proxyPort set HTTP_PROXY_USER=user set HTTP_PROXY_PASS=xxxxxxxx set HTTPS_PROXY=http://proxyServer:proxyPort set HTTPS_PROXY_USER=user set HTTPS_PROXY_PASS=xxxxxxxx {% endhighlight %}
